About Me

I've been programming since around 2012 for the Android Platform, recently, I've expanded to HTML, Dart, and CSS for the web. I love to use Open Source code, and give Open Source code through sites like GitHub, and I hope to one day have a large impact on the world through code.

I've been interested in coding since I my first , and the introduction of Android and iOS grew this intrest into a passion for code. Innovative companies such as Google have also influenced this passion, as I was really interested in seeing the amazing things that could be done simply with a few lines of code. Google's introduction of Material Design also gave me an intrest in UI and Icon design, which pushed my passion for code even further.

For a long while, I also had to deal with several mental challenges in my life. This, along with other factors lead me to want to one day use code to help out people that had to go through things like depression or anxiety, and show them that there are people that really do care.
